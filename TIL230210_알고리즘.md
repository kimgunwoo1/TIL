### 코드 배포

1. 클론
   ```
   git clone https://github.com/edu-neo/multi-it-ai-2.git
   ```

   

2. 깃에서 처음 파일 받아오기

   `````
   git pull origin master
   ```

   

3. 브랜치 생성하기
   ```
   git switch -c 김건우
   git branch (확인)
   git switch master (마스터로 변경)
   git branch -D 김건우 (지우기)
   ```

   

4. 파일 올리기
   ```
   git add .
   git commit -m 'p12928'
   git push origin 김건우
   ```
